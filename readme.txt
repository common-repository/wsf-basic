== Description ==

A plugin where users and visitors can post.

With the shortcode [wsf-load], it is a social flow plugin that you can easily use on any page you want.

= Privacy notices =

With the default configuration, this plugin, in itself, does not:

* track users by stealth;
* send any data to external servers;
* use cookies.

= Translations =

You can easily translate with Loco translate.

== Installation ==

1. Upload the entire `wsf-basic` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).
The plugin is easy to use. There is no setting etc. The page where it adds the [wsf-load] shortcode can also be run.
